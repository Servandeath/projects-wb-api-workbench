# Архитектура проекта

WB API Workbench состоит из нескольких слоёв:

## GUI layer

- окно выбора роли;
- менеджер API-ключей;
- тестер API-запросов;
- просмотр JSON;
- импорт локальных выгрузок.

## Core layer

- хранение ключей;
- права доступа;
- HTTP-клиент;
- обработка ошибок;
- логирование.

## WB API layer

- API-клиент;
- группы методов;
- JSON-схемы;
- загрузка OpenAPI-документации.

## Database layer

- SQLite;
- SQLAlchemy;
- история запросов;
- метаданные API-ключей;
- сохранение raw JSON.
